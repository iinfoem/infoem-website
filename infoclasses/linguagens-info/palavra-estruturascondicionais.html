Diariamente, nos comunicamos com pessoas de diversas maneiras, com a comunicação em inúmeras presenças cotidianas. 


Quase sempre nos deparamos com momentos que você deve fazer uma escolha entre duas ou mais opções, de uma forma bem simples, podemos dizer estou querendo fazer algo usamos nosso poder de escolha para tomar uma decisão, para isso você se baseia em uma condição. 


O que é uma estrutura de condições? 


De modo análogo ao que foi proposto anteriormente, os computadores em determinados momentos do código devem usar seu poder de escolha, para avaliar determinadas opções e seguir naquela que for coincidente com a condição anteriormente dita. 


Exemplo em JavaScript: 

prompt (x); if (x<5) {console.log("Sim");} 


O programa acima quando feito corretamente, ainda falta partes, pois não estamos para julgar conforme funcionalidade, mas a estrutura condicional em si. If(se) representa, se a condição (x<5) for verdadeira então faça o comando, caso não seja, não faça. 

Desse modo, o usuário irá dizer um número e se for menor que cinco ele devolverá a mensagem Sim! 


Obs: o comando prompt, representa, leia isso, ou seja, leia x.
