Aqui, quero trazer de uma forma simples, estruturas de repetição e condições. Como funciona a estrutura de condições? 


if (condição 1) { //pode ser x>=5 por exemplo comando //pode ser console.log por exemplo } 


Aqui temos a base de uma estrutura de repetição, nela possuímos: “if” traduzindo “se” representa que o programa deverá fazer uma escolha, tomar uma decisão. Entre parênteses, encontramos a condição. Entre chaves, encontramos um comando que só será realizado, se a condição for verdadeira. 


if (condição 1 and/or condição 2) { comando } else { comando } 


Aqui temos um programa de estrutura de condições um pouco mais completo, if ‘se’ a grande diferença está na condição, temos um operador lógico aqui, como visto em outro post, o operador and representa que o comando será realizado se ambas as condições forem verdadeiras. O operador representa que o comando será realizado se uma das condições forem verdadeiras. Caso, o if não seja realizado, o comando do ele será realizado. 


Como funciona a estrutura de repetições? 


while (condição) { comando } 


Aqui temos a base de uma estrutura de repetição. Enquanto a condição for verdadeira, realize o comando. Condições estruturais de um programa e a noções de matemática básica. Note que ambas as estruturas, requerem condições, para que tais condições sejam efetuadas, é necessário um conhecimento de operadores, operações e conectivos.
